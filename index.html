Matematică

Radicali de ordinul n

Definiție: Radicalul de ordin n dintr-un număr a este numărul b astfel încât bⁿ = a.

Proprietăți:

  • ⁿ√(a·b) = ⁿ√a · ⁿ√b
  • ⁿ√(a/b) = ⁿ√a / ⁿ√b (b ≠ 0)
  • (ⁿ√a)ᵐ = ⁿ√(aᵐ)
  • ⁿ√ᵐ√a = ᵐⁿ√a

Exemplu: √50/√2 = √(50/2) = √25 = 5

Logaritmi

Definiție: logₐb = c ⇔ aᶜ = b, unde a > 0, a ≠ 1, b > 0.

Proprietăți:

  • logₐ(x·y) = logₐx + logₐy
  • logₐ(x/y) = logₐx - logₐy
  • logₐxᵖ = p·logₐx
  • logₐb = 1/(log_b a)
  • Schimbarea bazei: logₐb = log_c b / log_c a

Exemplu: log₂8 + log₂4 = 3 + 2 = 5

Numere complexe

Forma algebrică: z = a + bi, unde i² = -1

  • Modul: |z| = √(a² + b²)
  • Conjugat: z̅ = a - bi
  • Forma trigonometrică: z = r(cosθ + i sinθ), unde r = |z|, θ = arg(z)

Exemplu: Pentru z = 3 + 4i: |z| = √(3² + 4²) = 5

Funcții injective, surjective, bijective

  • Injectivă: f(x₁) = f(x₂) ⇒ x₁ = x₂ (orice valoare din codomeniu are cel mult o preimagine)
  • Surjectivă: ∀y∈B, ∃x∈A astfel încât f(x) = y (codomeniul = imaginea)
  • Bijectivă: Este atât injectivă cât și surjectivă ⇒ inversabilă

Exemplu: f: R→R, f(x) = 2x+1 este bijectivă

Informatică

Operatori în C++

Operatori aritmetici: +, -, *, /, % (modulo)

Operatori de atribuire: =, +=, -=, *=, /=

Operatori de comparație: ==, !=, >, <, >=, <=

Operatori logici: && (și), || (sau), ! (not)

Exemplu: 5 + 3 * 2 = 11 (înmulțirea are prioritate)

Structuri decizionale (if)

Sintaxa:

if (condiție) {
    // instrucțiuni dacă condiția este adevărată
} else {
    // instrucțiuni dacă condiția este falsă
}

Exemplu:

int n = 5;
if (n % 2 == 0) {
    cout << "Par";
} else {
    cout << "Impar";
}

Structuri repetitive

while: Se execută cât timp condiția este adevărată

int i = 1;
while (i <= 5) {
    cout << i << " ";
    i++;
}

for: Pentru un număr cunoscut de pași

for(int i = 1; i <= 5; i++) {
    cout << i << " ";
}

do...while: Se execută cel puțin o dată

int x = 5;
do {
    cout << x << " ";
    x--;
} while(x > 2);

Engleză

Vocabular

Haine (Clothes): coat, jacket, scarf, gloves, hat, sweater

Caracteristici oameni (Personality): kind, friendly, confident, brave, attractive

Locuri în Veneția: canals, buildings, museums, festivals

Exemple:

  • She is very kind.
  • He is wearing a warm coat.
  • Venice has many famous canals.

Timpul viitor (Future Tense)

Future Simple (will): Pentru decizii spontane, predicții

  • I will travel to Italy next year.
  • She will not forget your birthday.

Going to: Pentru planuri, intenții

  • Look at the clouds! It is going to rain.

Future Continuous: Pentru acțiuni în desfășurare la un moment viitor

  • Tomorrow at 8 PM, I will be studying.

Question Tags

Folosite pentru a cere confirmare. Regulă: propoziție afirmativă → tag negativ și invers.

  • You like winter festivals, don't you?
  • He is not coming today, is he?
  • She can speak English, can't she?
  • They are coming later, aren't they?

Conditional Sentences

Type 1 (Real): If + present, will + verb

  • If it snows, we will stay at home.
  • If you study, you will pass the test.

Type 2 (Unreal present): If + past simple, would + verb

  • If I were you, I would not worry.

Type 3 (Unreal past): If + past perfect, would have + verb

  • If I had known about the festival, I would have gone.

Fizică - Termodinamică

Transformări termodinamice

Gazul ideal: pV = nRT (ecuația de stare)

Transformări:

  • Izotermă: T = constant ⇒ pV = constant
  • Izobară: p = constant ⇒ V/T = constant
  • Izocoră: V = constant ⇒ p/T = constant
  • Adiabată: Q = 0 ⇒ pVᵞ = constant, unde γ = Cₚ/Cᵥ

Lucrul mecanic și energia internă

Lucru mecanic (L):

  • Izobar: L = p·ΔV
  • Izoterm: L = nRT·ln(V₂/V₁)
  • General: L = ∫p dV (aria de sub curbă în diagrama p-V)

Energie internă (U): ΔU = Q - L (Primul principiu al termodinamicii)

  • Gaz monoatomic: U = (3/2)nRT
  • Gaz diatomic: U = (5/2)nRT

Căldura (Q): Q = nCΔT

Cicluri termodinamice

Definiție: Succesiune de transformări după care gazul revine în starea inițială.

Lucru mecanic pe ciclu: L_ciclu = aria ciclului în diagrama p-V

Variația energiei interne pe ciclu: ΔU_ciclu = 0

Randament: η = L / Q₁, unde Q₁ este căldura primită
